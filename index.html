Key Financial Formulas

Net Present Value (NPV): \(\; \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \;\)

Compound Annual Growth Rate (CAGR): \(\; \mathrm{CAGR} = \left(\frac{V_{\text{end}}}{V_{\text{start}}}\right)^{\tfrac{1}{n}} - 1 \;\)

Sharpe Ratio: \(\; S = \frac{E[R_p]-R_f}{\sigma_p} \;\)

Scenario Model Snippet (Python)

import numpy as np

# Discounted cash flow (illustrative)
def npv(rate, cashflows):
    return sum(cf / ((1 + rate) ** t) for t, cf in enumerate(cashflows, start=1))

rate = 0.11
cashflows = [420, 510, 590, 640]
print("NPV:", round(npv(rate, cashflows) - 1000, 2))
